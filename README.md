# California Housing Price Prediction using Deep Neural Networks

This project implements a deep neural network regression model to predict median house values in California using the California Housing dataset. Built with TensorFlow 1.x Estimator API, it demonstrates end-to-end machine learning workflow including data preprocessing, feature engineering, model training with hyperparameter tuning, and evaluation.

## 📋 Overview

The model predicts housing prices (in thousands of dollars) based on various features such as location, population demographics, and housing characteristics. The project explores two different neural network architectures and hyperparameter configurations to optimize prediction accuracy.
